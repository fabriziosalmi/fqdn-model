# Installation

## Prerequisites

- Python 3.8+
- pip

## Quick Start

1. Clone the repository:
   ```bash
   git clone https://github.com/fabriziosalmi/fqdn-model.git
   cd fqdn-model
   ```

2. Create a virtual environment:
   ```bash
   python3 -m venv venv
   source venv/bin/activate
   ```

3. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
